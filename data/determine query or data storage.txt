Determine Query or Data Storage
Main Success Scenario:
  1. How much drive and ram space is available
  2. How frequently does the data change
  3. Does the logic of the query need to be easily exposed
  4. Do you have permission to insert into a table
  5. Do you have a query run time limit
  6. Do you need to control the time the query is executed
  7. How frequently does the query change
  8. Does the result need to be joined on
  9. Is maintaining the query more important that maintaining the data
  10. Is the query only allowed to be in certain objects
  11. Result does not need to be saved in a table
